# Project Brief: Voksel Interactive Profile

## Tujuan

Membuat web profil interaktif PT Voksel Electric Tbk untuk kebutuhan magang/kuliah praktik. Web harus mudah dibuka, tampil profesional, dan bisa menjawab pertanyaan dasar melalui chatbot lokal.

## Pengguna

- Mahasiswa magang.
- Pembimbing perusahaan.
- Pengunjung yang ingin mengenal Voksel secara cepat.

## Fitur Saat Ini

- Profil perusahaan.
- Linimasa singkat.
- Bagian konteks Bogor untuk kebutuhan magang lokal.
- Kartu lini produk.
- Slot foto produk dan profil perusahaan.
- Tabel data perusahaan.
- Chatbot lokal berbasis kata kunci dan dataset.
- Tempat dataset terpisah di `data/voksel-dataset.js`.
- Section Keuangan & Investor untuk data publik sementara, laporan keuangan, dan tinjauan keuangan.
- Animasi visual jaringan kabel.

## Data Yang Perlu Dikonfirmasi

- Alamat detail unit atau kegiatan Voksel di Bogor.
- Data internal yang boleh ditampilkan publik.
- Struktur organisasi atau unit kerja tempat magang.
- Foto resmi, logo resmi, atau aset visual yang boleh dipakai.
- Data penjualan, karyawan, dan produksi jika perusahaan mengizinkan.
- Angka laporan keuangan rinci seperti aset, liabilitas, ekuitas, laba rugi, dan arus kas dari dokumen resmi perusahaan.

## Ide Pengembangan

- Chatbot tersambung ke API atau database.
- Dashboard laporan produksi atau informasi umum yang sudah diizinkan.
- Halaman admin untuk mengubah data tanpa mengedit kode.
- Integrasi dokumen PDF profil perusahaan atau laporan tahunan.
