# Folder Agent Proyek Voksel

Folder `agents` dipakai sebagai tempat catatan kerja untuk asisten atau pengembang yang membantu proyek ini. Isinya bukan bagian tampilan web utama, tetapi membantu menjaga arah proyek tetap jelas.

## Isi Folder

- `README.md`: penjelasan fungsi folder `agents`.
- `project-brief.md`: ringkasan tujuan proyek, fitur, data, dan ide pengembangan.
- `../data/voksel-dataset.js`: tempat dataset yang dipakai chatbot dan tampilan data.

## Fungsi

- Menyimpan konteks proyek agar pengembangan berikutnya tidak mulai dari nol.
- Mencatat batasan data, terutama data lokal Bogor yang perlu dikonfirmasi dari pembimbing perusahaan.
- Membantu presentasi karena kamu bisa menjelaskan bahwa proyek ini disiapkan dengan dokumentasi kerja yang rapi.

## Catatan

Folder `.agents` yang tersembunyi tidak saya ubah karena akses tulisnya dibatasi oleh sistem. Karena itu dokumentasi proyek dibuat di folder `agents` biasa.
