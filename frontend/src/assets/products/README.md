# Foto Produk

Folder ini disiapkan untuk foto produk Voksel.

Ganti file placeholder berikut dengan foto asli jika sudah mendapat izin perusahaan:

- `power-cable.svg`
- `telecommunication-cable.svg`
- `fiber-optic-cable.svg`
- `bare-conductor.svg`
- `special-cable.svg`

Nama file boleh tetap sama agar halaman otomatis memakai foto baru tanpa perlu mengubah kode.
